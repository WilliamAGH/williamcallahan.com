## Part 1: Setting Up Global JAVA_HOME

Here's how to set up `JAVA_HOME` globally on macOS and Windows, ensuring your system is ready for Java applications.

1. **Download Java SDK**: Visit the [Oracle website](https://www.oracle.com/java/technologies/downloads/) to download the latest Java SDK.

### For macOS

2. **Verify Java Version**: Open your terminal and run:
   ```bash
   java --version
   ```
   Ensure you have the correct version installed.

3. **Set JAVA_HOME**: Edit your `.zshrc` file:
   ```bash
   nano ~/.zshrc
   ```
   Then type:
   ```bash
   export JAVA_HOME=$(/usr/libexec/java_home)
   ```
   Type this to apply changes:
   ```bash
   source ~/.zshrc
   ```

But the GUI should let you set it via a dropdown menu.

---
### For Windows 10/11

2. **Set JAVA_HOME**: Open **System Properties** (Win + Pause/Break), go to **Advanced system settings**, and under **Environment Variables**, add:
   - **Variable name**: `JAVA_HOME`
   - **Variable value**: `C:\Program Files\Java\jdk-23.0.2` (or the path to the Java SDK you downloaded if different)

3. NA for Windows

---
4. **Configure IntelliJ IDEA**: Go to **File > Project Structure > SDKs** and ensure the correct JDK is selected. Set `JAVA_HOME` globally under **Preferences > Appearance & Behavior > Path Variables**:
   - **Name**: `JAVA_HOME`
   - **macOS Value**: `/Users/<username>/Library/Java/JavaVirtualMachines/openjdk-23.0.2/Contents/Home` (or the path to the Java SDK you downloaded if different)
   - **Windows Value**: `C:\Program Files\Java\jdk-23.0.2` (or the path to the Java SDK you downloaded if different)

## Part 2: Project-Level Settings for JavaFX

Now you'll use JavaFX in your project.

1. **Integrated Setup in IntelliJ IDEA and VS Code**: Both IntelliJ IDEA and VS Code offer seamless onboarding for JavaFX projects. When creating a new project, select Java > JavaFX from the project templates, and the IDE will automatically configure the necessary settings.

2. **Alternative - Download JavaFX**: If needed, visit the [Gluon website](https://gluonhq.com/products/javafx/) to download the latest LTS version.

3. **Place the SDK**: You'll need to place the JavaFX SDK in a path under something like `~/Library/Java/JavaVirtualMachines/openjdk-23.0.2/Contents/Home/lib/ext`

## Adding JavaFX to an Existing Java Project

### In IntelliJ IDEA

1. **Open Project Structure**: Go to **File > Project Structure**.
2. **Select Modules**: Click on the **Modules** tab.
3. **Add JavaFX**: Click on the **Dependencies** tab within the selected module. Use the **+** button to add a new library, and select **JavaFX** from the available options. IntelliJ will automatically configure the necessary settings.

### In VS Code

1. **Open Command Palette**: Press `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` (Windows/Linux) and type `Java: Add Libraries to Classpath`.
2. **Select JavaFX**: From the list of available libraries, select **JavaFX**. VS Code will handle the configuration for you.
